Portfolio

Gym Program Viewer

Webapp that allows user to create workout programs. User can view exercises, reps and sets from previously made workouts for each day.

Technologies: Java - Spring Boot, React.js

Tourify

Class Project for CMPT 470 (Web-based Information Systems). Webapp for community organized cycling races with real-time GPS Coordinates.

Technologies: Node.js, Express, React.js, Firebase
*Source code restricted to SFU hosted Gitlab

Social Gaming Platform

Class Project for CMPT 373 (Software Development Methods). Designed a platform for users to host server for players to create game rooms and play selected text base games with their friends.

Technologies: C++
*Source code restricted to SFU hosted Gitlab

Connect 4 AI

Class project for CMPT 310 (Artificial Intelligence Survey). Created an AI program to play Connect 4 using Monte Carlo Tree with heuristics.


Technologies: C++ 

FilmStudies

Class project for CMPT 276 (Introduction to Software Engineering). Created a webapp to search for movies and to allow users to read and write reviews for them.


Technologies: Java - Spring Boot, React.js

Course Watcher

Class project for CMPT 213 (Object Oriented Design in Java). Webapp that allows user to view class enrollment and add new classes to track.


Technologies: Java - Spring Boot 
*Source code restricted to SFU hosted Gitlab
